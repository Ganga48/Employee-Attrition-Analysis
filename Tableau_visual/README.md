# Employee Attrition Analysis - Tableau Dashboards

## 📌 Overview
This section of the project contains interactive Tableau dashboards built to visualize employee attrition trends and provide actionable insights for HR teams. The dashboards transform raw analysis into clear, dynamic visuals that highlight key drivers of turnover.

## 🎯 Objectives
- Present attrition rates across departments, job roles, and demographics.
- Visualize correlations between employee performance, satisfaction, and attrition.
- Provide HR managers with interactive tools to explore workforce data.
- Support decision-making with clear, data-driven visuals.

## 🛠 Tools & Technologies
- **Tableau Public** → Sharing and publishing dashboards

## 📊 Key Dashboards
- **Attrition Overview** → High-level attrition rates and trends
- **Demographic Analysis** → Attrition by age, gender, education, and marital status
- **Job Role Insights** → Attrition patterns across different job categories
- **Performance & Satisfaction** → Impact of performance ratings and satisfaction scores on attrition



## 📈 Deliverables
- Interactive dashboards for HR analysis
- Visual reports highlighting attrition drivers
- Exported dashboard images for presentations
